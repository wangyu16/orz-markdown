Features

orz-markdown is a markdown-it build with a uniform {{…}} plugin syntax, ::: containers, math, diagrams, and themes — the engine all three document tools render with.

Everything standard, plus extensions

GitHub-flavored Markdown — headings, bold/italic, lists, tables (with alignment), blockquotes, fenced code with highlighting, task lists, footnotes — plus ==mark==, ++ins++, ~~del~~, ~sub~, ^sup^, and image sizing ![alt](url =100x80).

Custom plugins — {{name …}}

One uniform syntax. Single-line {{name body}} or multi-line {{name⏎body⏎}}.

PluginExampleDoes
span / sp{{sp[red] text}}colored / badged inline spans
emoji / em{{emoji wave}}Unicode emoji by name
qr{{qr https://orz.how}}an inline QR code (click to enlarge)
youtube / yt{{yt dQw4w9WgXcQ}}a responsive embed
mermaid / mm{{mm⏎graph LR⏎A-->B⏎}}diagrams (mermaid.js)
smiles / sm{{smiles C1=CC=CC=C1}}chemical structures
chart{{chart⏎type: bar⏎…}}bar/line/pie charts (Chart.js)
toc{{toc 2,3}}auto table of contents
attrs# Title{{attrs[#id]}}add id/class/attributes

Containers — ::: name … :::

Semantic callouts and layout blocks:

::: info        ::: success      ::: warning      ::: danger
::: left 30%    ::: right        ::: center

:::: cols 1 2        :::: tabs          ::: spoil Click to reveal
::: col              ::: tab Label      hidden content
:::                  :::                :::
::::                 ::::

A space after ::: is required; nesting uses more colons on the outer block. Any ::: ClassName not reserved becomes a <div class="ClassName">.

Math

KaTeX (with mhchem for chemistry): inline $E=mc^2$ and display $$\int_0^1 x^2\,dx$$, rendered at parse time.

Themes

Ten ready-made CSS themes — light/dark, academic/neat/playful/beige — that style the parser's full output (containers, tabs, math, diagrams, QR overlays, print). Rendered HTML lives in an .markdown-body wrapper.

Copy as Markdown

With the browser runtime loaded, selecting rendered content and copying it puts Markdown source on the clipboard — not HTML. Tables, lists, math, code, and even generated constructs (a copied QR yields {{qr …}}) round-trip back to source.
