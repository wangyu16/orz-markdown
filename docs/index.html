orz-markdown

A customized Markdown parser — and a family of self-contained, editable HTML document tools that travel in a single file.

The parser

orz-markdown is a markdown-it build with custom {{…}} plugins (mermaid, qr, youtube, smiles, toc, span, charts…), ::: containers, KaTeX math, and bundled themes — the engine all three tools render with. GitHub · npm

Verify a document

These files are programs: opening one runs the JavaScript inside it. Before you trust a file from someone else, check it — the verifier hashes its embedded framework against the genuine published packages and scans the content for embedded code. It runs entirely in your browser.

Good to know

  • Editing & in-place Save need a Chromium-based browser (File System Access); viewing works everywhere.
  • Treat these files like downloaded programs — only open or edit ones from a source you trust.
  • When a newer framework ships, the in-file editor offers a one-click, opt-in update.
